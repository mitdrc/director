Keyboard Shortcuts
==================

This documents the shortcuts available in Director and related applications.

Director
--------

  * :kbd:`Ctrl` + :kbd:`Q` -- quits the application
  * :kbd:`Ctrl` + :kbd:`O` -- load a polyData data file (vtp, ply)
  * :kbd:`Ctrl` + :kbd:`S` -- save currently selected object-model polyData as a data file
  * :kbd:`Ctrl` + :kbd:`B` -- opens bot-spy (LCM traffic inspection tool)
  * :kbd:`Ctrl` + :kbd:`I` -- opens signal-scope
  * :kbd:`Ctrl` + :kbd:`M` -- opens the Multisense panel
  * :kbd:`Ctrl` + :kbd:`F` -- opens the footsteps panel
  * :kbd:`F8` -- opens the Python console
  * :kbd:`F9` -- opens the Matlab console
  * :kbd:`F10` -- opens the task panel
  * Footsteps Driver, Playback Panel:

    * :kbd:`Ctrl` + :kbd:`Enter` -- executes the footstep plan

  * Teleop Panel:

    * :kbd:`Ctrl` + :kbd:`Shift` + :kbd:`S` -- plan standing posture
    * :kbd:`Ctrl` + :kbd:`Shift` + :kbd:`N` -- plan nominal posture
    * :kbd:`Ctrl` + :kbd:`Shift` + :kbd:`L` -- plan pregrasp with left hand
    * :kbd:`Ctrl` + :kbd:`Shift` + :kbd:`R` -- plan pregrasp with right hand
    * :kbd:`Ctrl` + :kbd:`Shift` + :kbd:`P` -- equivalent to pressing the :kbd:`Plan` button

  * Python Console:

    * :kbd:`Ctrl` + :kbd:`D` / :kbd:`Ctrl` + :kbd:`W` -- closes the python console


Console App
-----------

  * :kbd:`Ctrl` + :kbd:`Q` -- quits the application
  * :kbd:`F1` -- shows the object model
  * :kbd:`F8` -- opens the Python console


Drake Visualizer
----------------

  * :kbd:`Ctrl` + :kbd:`Q` -- quits the application
  * :kbd:`F1` -- shows the object model
  * :kbd:`F2` -- toggles the screen grabber
  * :kbd:`F3` -- toggles the camera bookmarks
  * :kbd:`F8` -- opens the Python console
